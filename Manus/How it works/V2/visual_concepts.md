# Visual Concepts Extracted from Relevic Deck

## Key Visual Elements to Implement for Grona

### 1. **3-Step Workflow Diagram** (Page 6)
**Concept**: Simple horizontal flow with icons
- **Visual**: Three connected cards showing: "Enter a prompt" → "Preview results" → "Deploy instantly"
- **Icons**: Yellow chat bubble, pink target/preview icon, yellow rocket
- **Style**: Dark cards with colored icons, connected by arrows
- **Application for Grona**: Simplify the current 4-step process or create a high-level overview

### 2. **Timeline/Journey Diagram** (Page 8)
**Concept**: Horizontal timeline with phases
- **Visual**: Three phases connected by a gradient line (yellow to pink to yellow)
- **Phases**: Learn (Weeks 1-2) → Suggest & Deploy (Weeks 3-4) → Review (Week 5)
- **Icons**: Magnifying glass, robot/AI, checklist
- **Style**: Circular icons on a horizontal line with descriptions below
- **Application for Grona**: Show the optimization cycle or customer journey

### 3. **Comparison Table** (Page 11)
**Concept**: Feature comparison matrix
- **Visual**: Table with checkmarks, X marks, and half-circles for partial features
- **Highlight**: Golden/yellow column for the product, darker columns for competitors
- **Features**: AI-First Design, No-Code Implementation, A/B Testing, etc.
- **Application for Grona**: Already implemented but could enhance with better visual hierarchy

### 4. **Circular Growth Flywheel** (Page 14)
**Concept**: Circular diagram showing interconnected strategies
- **Visual**: Center circle labeled "Growth Flywheel" with icons around the perimeter
- **Elements**: Direct Outreach, Free CRO Sprint, Agency Partner, Content Marketing, SEO, AI Demos
- **Style**: Gradient ring (pink to yellow) with icon badges
- **Application for Grona**: Show continuous optimization loop or AI agent collaboration

### 5. **Metric Cards with Icons** (Throughout deck)
**Concept**: Large numbers with supporting icons
- **Visual**: Big bold numbers with small circular icon badges
- **Examples**: 
  - "150+ Total Users" with people icon
  - "10 Paying Customers" with payment icon
  - "£250K SEIS Funding Raised" with chart icon
- **Style**: Yellow/gold icons on dark backgrounds
- **Application for Grona**: Stats section showing platform performance

### 6. **Feature Highlights Grid** (Page 7)
**Concept**: 2x2 or 2x3 grid of features with icons
- **Visual**: Large circular icons with feature titles and descriptions
- **Icons**: Pencil (Create with AI), People (Smart Personalization), Flask (A/B Testing), Brain (AI Insights)
- **Style**: Dark circular backgrounds with colored icons
- **Application for Grona**: Already partially implemented, could enhance

### 7. **Revenue/Growth Chart** (Page 15)
**Concept**: Dual-line chart showing growth trajectory
- **Visual**: Yellow line (Revenue) and Pink line (EBITDA) with data points
- **Style**: Clean, modern chart with gradient fill under lines
- **Metrics**: CAC, LTV, User Growth, Gross Margin shown as cards
- **Application for Grona**: Show conversion improvement metrics or ROI

### 8. **Donut Chart with Allocation** (Page 17)
**Concept**: Funding allocation visualization
- **Visual**: Colorful donut chart with legend
- **Colors**: Yellow, Pink, Purple, Blue segments
- **Application for Grona**: Could show time allocation of AI agents or resource distribution

## Design Principles from Relevic Deck

1. **Dark backgrounds** with high contrast text
2. **Yellow/Gold and Pink/Magenta** as primary accent colors
3. **Circular icon badges** for features and metrics
4. **Horizontal flow diagrams** with connecting arrows
5. **Gradient lines** for timelines and progress indicators
6. **Large, bold typography** for numbers and headlines
7. **Minimal, clean layouts** with ample whitespace
8. **Consistent icon style** - simple, modern, flat design
